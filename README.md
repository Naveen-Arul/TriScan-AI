# TriScan AI

TriScan AI is an advanced AI-powered platform that combines intelligent OCR, smart file comparison, and web content extraction in one unified interface.

## Features

- **Intelligent OCR**: Advanced optical character recognition for document processing
- **Smart File Comparison**: Compare documents and files with precision
- **Web Content Extraction**: Extract clean content from websites
- **Multi-mode Interface**: Switch between OCR, comparison, and scraping modes

## Technology Stack

- React with TypeScript
- Vite for fast development
- Tailwind CSS for styling
- shadcn/ui components
- React Router for navigation

## Getting Started

1. Clone the repository
2. Navigate to the frontend directory:
   ```
   cd frontend
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm run dev
   ```
